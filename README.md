# 🐍 Scripts Python

![Python](https://img.shields.io/badge/python-3670A0?style=for-the-badge&logo=python&logoColor=ffdd54)

Repositório de uma coleção de scripts utilitários (com interface gráfica) desenvolvidos em Python, com foco em automação de tarefas corriqueiras.
Cada script está organizado em sua própria pasta e pode ser executado de forma independente.

## 🛠️ Tecnologias

- Python 3
- PyAutoGUI
- Pynput
- Pillow
- CustomTkinter

## 📁 Estrutura do Repositório

Cada pasta desse repositório contém um script específico.

| Script | Descrição |
|---|---|
| [Auto Clicker](./Auto%20Clicker) | Automatiza cliques do mouse em intervalos configuráveis |
| [Automatizador de Banco de Dados](./Automatizador%20de%20Banco%20de%20Dados) | Automatiza operações de criação e inserção de dados em BD |
| [Gerador de QR Code](./Gerador%20de%20QR%20Code) | Gera QR Codes a partir de textos ou URLs informados |
| [Macro de Texto](./Macro%20de%20Texto) | Define atalhos de teclado em textos predefinidos |
| [Organizador de Pastas](./Organizador%20de%20Pastas) | Organiza arquivos de um diretório em subpastas por tipo/extensão |
| [Renomeador de Arquivos](./Renomeador%20de%20Arquivos) | Renomeia arquivos em lote seguindo um padrão definido |

### Dentro dos Projetos

```
Nome do Projeto/
├── app.py       # Arquivo principal do projeto
├── core.py      # Arquivo do funcionamento do projeto
└── gui.py       # Arquivo da interface gráfica do projeto
```

## 🚀 Como executar

1. Clone o repositório:
```bash
git clone https://github.com/rafaelbaratta/scripts-python.git
```

2. Acesse a pasta do script desejado:
```bash
cd scripts-python/<nome-do-script>
```

3. Instale as dependências (se houver):
```bash
pip install -r requirements.txt
```

4. Execute o script:
```bash
python app.py
```
